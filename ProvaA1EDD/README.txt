Engenharia de Software
Testes JUnit feitos por Leonardo Hassan Zeaim e Lucas Barros 


Os testes foram feitos com o JUnit pela ide IntelliJ. Applicados em uma applicação feita anteriormente na materia de Estrutura de dados, havia uma pilha e uma fila sendo utilizadas.

Podemos encontrar os arquivos necessarios para efetivar os testes no /lib
Os testes com detalhes estão no /src, juntamente com os arquivos em java onde se encontra a applicação.
